# Kalkulator Bunga Sederhana

Proyek ini menjelaskan cara menghitung bunga sederhana.

## Rumus Bunga Sederhana
Bunga dihitung dengan rumus:
`Bunga = (P * R * T) / 100`

## Keterangan Variabel
* **P (Principal)**: Jumlah pinjaman atau investasi awal.
* **R (Rate)**: Suku bunga per tahun (dalam persen).
* **T (Time)**: Jangka waktu dalam satuan tahun.

## Contoh Perhitungan
Jika P = 1.000.000, R = 5%, dan T = 2 tahun:
Hasil: (1.000.000 * 5 * 2) / 100 = 100.000.
