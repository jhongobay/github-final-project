# Kalkulator Bunga Sederhana

Proyek ini adalah aplikasi sederhana untuk menghitung bunga tetap (simple interest) berdasarkan pokok pinjaman, tingkat bunga, dan periode waktu.

## Rumus yang Digunakan
Aplikasi ini menggunakan rumus bunga sederhana standar:
**A = P(1 + rt)**

Keterangan:
- **A**: Jumlah total (pokok + bunga)
- **P**: Pokok pinjaman (Principal)
- **r**: Suku bunga tahunan (decimal)
- **t**: Waktu atau durasi (tahun)

## Cara Penggunaan
1. Masukkan jumlah pokok pinjaman.
2. Masukkan tingkat bunga tahunan (dalam persen).
3. Masukkan durasi waktu dalam tahun.
4. Klik "Hitung" untuk melihat total bunga dan jumlah akhir.
